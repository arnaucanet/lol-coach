🎯 LoL Coach

A personal companion web tool for League of Legends players who want to prepare, learn and improve.

View source on GitHub Powered by Riot Games API
👤

Summoner Profile

Type your Riot ID and see your ranked stats, top champion masteries, recent match history with detailed KDA and item builds, and detected duo partners you play with most often.

🎮

Live Game Companion

When you're in an active ranked game, see both teams enriched with each player's runes, summoner spells, current rank and mastery on the champion they're playing — all in one place before the loading screen ends.

📊

Champion Builds

Real build recommendations calculated from recent SoloQ Ranked matches of top players (Iron to Challenger), filterable by region or Global. Real win rates, pick rates, ban rates, item builds, runes, counters and synergies — no invented data.

🧠

Draft Coach

Given the 10 champions in a match (either manually or loaded from the live-game endpoint), get an item/rune/skill-order suggestion for your specific matchup. Uses only info visible before the loading screen ends.

Riot Games APIs Used

  • account-v1
  • summoner-v4
  • league-v4
  • league-exp-v4
  • champion-mastery-v4
  • match-v5
  • spectator-v5

Static assets (champion icons, item icons, runes, summoner spells, rank emblems) come from Data Dragon.

✅ How this tool behaves

  • Only shows information the player already has access to via the client, profile page or match history.
  • No augment or Arena win rates.
  • No MMR or ELO calculator.
  • No analysis of hidden players — only Riot IDs explicitly entered by the user.
  • No tracking of information hidden from the game client.
  • No monetization, no ads, no third-party scraping.
  • Internal rate limiter + multi-tier caching to respect Riot API infrastructure.
  • Full source code publicly available on GitHub for transparency.
